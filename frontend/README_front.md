# SIGNA — Sistema Integrado de Gestão de Normativas Aeronáuticas

Sistema interno da Akaer para gestão e consulta de normativas aeronáuticas.

## Equipe

Desenvolvido por **JanoSys Technologies**

## Tecnologias

- React
- Vite
- CSS puro

## Como rodar o projeto
```bash
npm install
npm run dev
```

## Estrutura do projeto
```
src/
├── auth/
├── components/
├── pages/
│   ├── MindMap.jsx
│   ├── Normativas.jsx
│   ├── Notas.jsx
│   └── Usuarios.jsx
├── styles/
├── App.tsx
├── App.css
├── index.css
└── main.tsx
```

## Branches

- `main` — produção
- `dev` — integração geral
- `front` — integração do front
- `back` — inegração do back
- `dev-front-login` — tela de login
- `dev-front-normativas` — tela de normativas
- `dev-front-notas` — tela de notas
- `dev-front-usuarios` — tela de usuários
